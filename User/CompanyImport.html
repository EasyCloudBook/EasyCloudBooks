Company Import

Import many companies into Easycloudbooks at once, saving your time when you're setting up your practice. You can import Companies using a template XLS file, or a XLXS file. If you already have companies in Easycloudbooks, you can import more.

Which are the Mandatory Fields in company import ?

Company Name :- it is a Compulsory field. Company name should be atleast 10 charecters long. (already existing companies will not be imported).
Industry Type :- Voluntary Field (Note Industry type should be as per easycloudbooks supported industry otherwise it will display default industry type as OTHER.
Legal Structure : Voluntary field (Note Legal structure field should be as per easycloudbooks supported legal structure otherwise it will display OTHER.
Note : Voluntary Field in which you can mention small text description about company.

Which file format supported by Easycloudbooks?

.xls or .xlxs formats supported by system.

What is the maximum size of file which is to be imported and how many companies are imported at a time?

1000 companies are imported at a time and Up to 2 mb size, file can be imported.

Any report is generated after completion of Company import?

Yes, Report is generated after completion of import which contain
1.Successfully Imported Companies Count
2.Already Existed Companies Count
3.Error Companies Count
4.File Rejection Message when file is not in proper format.
